WordPress使用及Tag Cloud插件

发表于 2012-02-09

上几天自己买了个域名想搭建个人博客，很多人都选择了WordPress,所以想自己也学习一下

由于以前有接触过CMS后台管理系统，WordPress的后台管理也差不多。

WordPress最大的好处就是开源，然后有很多的插件可以下载。

这里推荐一个网站，专门讲解WordPress，而且非常详细，很适合初学者。

http://www.wpcourse.com

当然如果你以前有经验的话，这个网站里面的教学视频根本可以不用看，你可以去看一下里面推荐的一些wordpress插件

比如垃圾评论的处理，编辑器的改进，语法高亮，翻页设置，各种小工具等等

可以很好的改进您的网页功能和外观，方便您的管理

这里还想讲一下WP-Cumulus这个插件。由于我的一个朋友对他非常感兴趣，导致我也对他很感兴趣了。

这是一个flash tag cloud插件，鼠标消息可以控制3D标签做圆运动，整个像个球一样，动态的感觉很棒。

一开始这个插件是在wordpress上的，因为非常受欢迎，所以很快便有了js版本以及各种版本

自己还找到了一个类似功能的插件：3D Tag Cloud 不过感觉它的算法就比较简单了

下面这个链接是作者的个人网站对这个插件的介绍：

http://www.roytanck.com/2008/03/06/wordpress-plugin-wp-cumulus-flash-based-tag-cloud/

至于这个插件具体的实现方法的代码，可以参考下面这个链接:

http://riashanghai.com/zh-hant/node/72

这里作者没有给出标签做圆运动轨迹，方向速度大小变化的关键的代码，作者是通过反编译得出的算法。
