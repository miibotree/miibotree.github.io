What does CRLF means?

发表于 2014-04-04

今天上Unix课程的时候老师讲了一个比较好玩的东西，关于CRLF的含义。

讲到echo命令的时候，尝试man echo:

Echo the STRING(s) to standard output. 把字符串输送标准输出上

-n     do not output the trailing newline 不追加新行，即光标停留在行尾

-e     enable interpretation of backslash escapes 启用转义字符

-E     disable interpretation of backslash escapes (default)

&#8211;help display this help and exit

&#8211;version

       output version information and exit

If -e is in effect, the following sequences are recognized:

\\     backslash

\a     alert (BEL)

\b     backspace

\c     produce no further output

\e     escape

\f     form feed

\n     Carriage return and a line feed (new line)

\r     carriage return

\t     horizontal tab

\v     vertical tab

 

这里着重看一下\r 和\n 两个转义字符的英文解释：

\r carriage return

carriage 的中文意思是马车，return的中文意思是返回。所以carriage return的意思就是回车。

（原来回车是这么来的，瞬间觉得吓尿了有木有）

那么英文里面的carriage return 为什么能表示回车呢？这就跟打字机的结构有关啦！

[

http://oztypewriter.blogspot.com/2011/06/on-this-day-in-typewriter-history-xxii.html

](http://oztypewriter.blogspot.com/2011/06/on-this-day-in-typewriter-history-xxii.html)

打字机内部有一个滚筒一样的轴，然后纸张是卷起来绕在轴的外围一圈。打字的时候呢不同的按键伸出不同的像刻印章一样的东西啪嗒敲在纸张上，当然中间还隔了一层墨纸所以一个字就打出来了。

这里要注意的是不论按哪一个按键，按键敲的都是同一个位置，但是每次敲一个按键，纸张都会自动向左移动一点（通过轴来带动）。这样子不断移动纸张就可以打出一行字。

但是打字机不会自动换行的，当纸张移动到边缘的时候，需要把轴再次移到原来的位置，这个过程就是carriage return。但是请注意这个过程只是回到了本行的行首，也就是说还是在原来那行！！

 

接下来再来理解一下line feed的含义。feed是喂养的意思。为什么从一行的行首移动到下一行的行首的过程叫做line feed呢？同样还是跟打字机的结构有关。刚才的carriage return是轴的左右水平的移动。前面说了纸是贴在轴的外围的（可以想象成一个水平放的圆柱体）。所以line feed的过程其实就是轴转了一个很小的角度的过程。由于这个轴的截面有点类似于齿轮的形状，所以轴转动的过程很像动物张开嘴巴吃东西的样子，所以英语里使用了feed这个单词。

 

在解释了carriage return line feed这两个词组之后，我们再回过头来看看\r \n的解释:

\r carriage return

\n Carriage return and a line feed (new line)

这里我们看到，Unix/Linux把 \n,也就是new line 解释为先进行carriage return这个动作，再进行line feed动作。

这个过程让我想到了操作系统在硬盘定位的时候执行的也是类似的过程。先找磁道，再找扇区。原理是一样的。

So CRLF exactly means carriage return and line feed.
