A new start!

发表于 2013-10-23

想学网络，于是把windows给格掉了。

还是装fedora19，gnome3的桌面。

想把桌面搞的花里胡哨一点玩起来比较开心。

1.gnome 配置

https://wiki.archlinux.org/index.php/GNOME_%28%E7%AE%80%E4%BD%93%E4%B8%AD%E6%96%87%29

https://wiki.gnome.org/GnomeShell/CheatSheet

换了主题之后感觉爽多了

2.goagent

https://code.google.com/p/goagent/wiki/GoAgent_Linux#%E5%AE%89%E8%A3%85gtk%E6%89%98%E7%9B%98%E6%89%80%%9C%80%E6%A8%A1%E7%BB%84

[https://code.google.com/p/goagent/wiki/InstallGuide#%E9%99%84%EF%BC%9A%E6%B5%8F%E8%A7%88%E5%99%A8%E8%AE%BE%E7%BD%AE%E6%96%B9%E6%B3%95

至于注册google app就不多说了

下面添加一项自启动让goagent开机自己跑起来

gnome-session-properties

这里add一项自启动，在command里面加入 python /$PATH/proxy.py

注意权限问题，我一开始以root修改的结果没效，然后以普通用户修改之后就可以了
