SML & Emacs

发表于 2013-01-27

闲着没事情干，在class.coursera.org上面修了一门program language。当做是练习英语听力。

感觉课程量很大，而且存在语言障碍，听起来有点吃力…

算了，还是得一步步慢慢来。

初步感受了Emacs，跟vim相同的是都有很多快捷键，但是跟vim不同的是没有vim那样模式切换，需要频繁的按Esc，而是直接通过Ctrl 和Alt的功能键组合来实现。而且Emacs支持很多插件，SML完全集成在了Emacs里面，人家说Emacs就是一个操作系统那么强大。这几天体会了一下，果然很强大，跟vim伯仲之间，各有千秋。

vim的编辑效率很高，而Emacs集成各种插件，一半学Lisp估计都是用的Emacs吧。

简单看了看Emacs基本教程，顺便记几个快捷键

C-c C-s sml command

C-x C-f openfile

C-x C-c Close Emacs

C-d KillProcess

M-p repeat order

移动的时候

M 跟词有关

C 跟行有关

课程的学习就不打算写博客了，准备打算笔记记录就好了。如果碰到问题的话可以专门写博客。

看来还得学好英语，英语太重要了。
