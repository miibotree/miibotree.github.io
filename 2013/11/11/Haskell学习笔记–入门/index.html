Haskell学习笔记–入门

发表于 2013-11-11

看了下面这个Haskell趣学指南，觉得haskell挺好玩的。看这个网页有插画感觉超级友爱

http://fleurer-lee.com/lyah/chapters.htm

做一下学习笔记。

1. 在haskell中，每个函数和表达式都要返回一个结果

2. let a = 1 使用let来定义关键字

3.List

++合并操作 List [1,2,3,4] ++ [9,10,11,12] 注意List中的元素类型必须是一样的

++会遍历左边的所有元素

4. ：将左边的单个元素插入到右边的list中

5.若是要按照索引取得List中的元素，可以使用!!运算符，索引的下标为0。

6.

comprehension 集合

Haskell的函数编程的强大才刚刚开始体现。

| 右边的条件表达式，可以用逗号隔开。

我们来理解一下这句话：

x <- xs, 把xs列表的元素给x，

odd x 选取 x集合中的奇数

if x < 10 then “BOOM!” else “BANG!” 小于10的奇数元素为BOOM，大于10的奇数元素为BANG

7.

选取List元素中大写字母

8.Tuple 元组

fst返回一个序对的首项。

snd返回序对的尾项。

这两个函数只能用于二元组

9.

选取周长为24的直角三角形

10.

类型说明 从左边开始表示参数，最右边的表示返回值类型。中间用->符号连接起来

11.

在这里我们见到个新东西：=>符号。它左边的部分叫做类型约束。我们可以这样阅读这段类型声明：“相等函数取两个相同类型的值作为参数并返回一个布尔值，而这两个参数的类型同在Eq类之中（即类型约束）”

12.

fromIntegral :: (Num b, Integral a)

fromIntegral 将浮点数和整数相加。面对这种情况，我们就用fromIntegral (length [1,2,3,4]) + 3.2来解决。
